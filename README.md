## Getting Started

### Prerequisites

- Node.js 20 or higher
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Agent4343/grocery-weekly-planner-app.git
cd grocery-weekly-planner-app
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
Environment variables are optional for the demo. If you add integrations later, place them in a `.env.local` file at the project root.

4. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Available Scripts

- `npm run dev` - Start development server on port 3000
- `npm run build` - Build the application for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint for code quality
